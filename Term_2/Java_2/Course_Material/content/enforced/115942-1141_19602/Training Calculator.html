About the assignment

Graphical User Interfaces (GUIs) are the most common method with which users interact with an application.  Microsoft revolutionized GUI displays when they introduced Windows in 1985.  The introduction of Microsoft Windows brought the computer into the every day household, and made computers part of our modern life.

Computers can be found in everything today, from our desktops, laptops, cell phones, and PDAs, to watches, thermostats, and portable devices, such as bike computers.  A bike computer is a small device, about the size of a cookie, which attaches to the handlebars of a bicycle and is able to provide information to the cyclist such as current speed, time elapsed, distance traveled, average speed over time, and maximum speed.  The more advanced bike computers include GPS systems which also tell the altitude, slope of a hill, and current location.

The bike computer calculates speed and distance from magnetic sensors which are attached to the bicycle wheels and the pedal arms.  When the computer is installed, the cyclist programs the bike computer with the wheel size.  The computer counts the number of times the wheel turns and pedal cranks per second and, based upon the wheel size and leg speed ("cadence"), calculates the average speed.  The higher the gear, and the faster the cyclist is pedaling, the faster the average speed will be.

In this assignment, you will build a simple bike computer which will calculate average speed from the gears and pedal cadence.

The Frame

  • Create a project named BikeComputer.
  • Create two packages in the project:  velo and velo.gui.
  • Create a BikeComputer class in the velo package.  The BikeComputer class includes a main() method.
  • Create a BikeComputerFrame class in the velo.gui package.  The BikeComputerFrame class extends JFrame.
  • You only need two lines of code in the BikeComputer::main() method, one to instantiate the frame, and one to set it to visible.
  • The buttons must have functionality: 
    • when the user clicks Calculate, the program will display the average speed based upon front chainring, rear cog, and cadence.
    • when the user clicks Clear, the program will erase the cadence value and reset the average speed to 0.0kph.
    • when the user clicks Close, the program will terminate.
