Course Overview

Course Coverage

This course continues where PROG10082 left off and includes the following topics (not necessarily in this order):

  • Emphasizing professional and efficient program design.
  • Emphasizing proper use of OOP concepts:
    • Class Relationships, with an emphasis on Inheritance
    • Abstract classes and interfaces
    • Polymorphism
    • Packages
    • Developing documentation
  • String processing (review)
  • Exception Handling and Exception classes
  • Arrays:
    • Overview of Arrays, concepts, and applications using Arrays
    • Multi-dimensional arrays
    • Object Arrays and Arrays of GUI components
    • ArrayList class and Generics
  • File Input and Output:
    • Character streams used for reading and writing file data
    • The File class
    • Sequential files and random access files
    • File input and output
    • File processing - navigating records, record maintenance
  • Graphical User Interfaces using Swing:
    • GUI and Interface Design
    • Swing Components and their usage (e.g. text boxes, buttons, list boxes, menus, dialog boxes, etc)
    • Layout Managers
    • Event Handling
  • JAR files and creating a library of code
  • Application Deployment
  • JUnit Testing (DRAFT/Unofficial)

Course Evaluation

Your grade for this course is calculated in the following way:
Quizzes, Exercises: 10%
Assignments: 15%
Final Project: 10%
Mid Term Exam (week 7): 30%
Final Exam (week 14): 35%

"Quizzes, Exercises" refers to in-class assignments, quizzes (which may or may not appear in the Calendar), and "spot-checks" of assignments or project work. "Assignments" refers to take-home exercises or homework. "Project" refers to a large project that will take place during the last third of the course. You can either do a pre-defined project or you can submit a proposal for your own project idea.
