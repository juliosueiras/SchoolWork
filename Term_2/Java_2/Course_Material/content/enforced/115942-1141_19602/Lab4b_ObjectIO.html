Lab 4(b):  Object I/O

Due: Monday, March 18, 2013 @ 11am

About this lab

In the second part of the input/output assignment, you will use Object input and output streams to write whole objects to a file and read them back.

Part 1: Build the model

Write a serializable class, called UserRegistration, which will be read from and written to an external file.  Your code must do the following:

  1. Create a class called UserRegistration.
  2. The class must implement the Serializable interface.
  3. The class must contain basic user registration fields:  a username (String), a user ID (int), the user's first name and last name (Strings), a password (String), and an email address (String)
  4. Build a constructor for the class which will, at minimum, set the username, userID, and password.
  5. Build getters and setters for all fields except the password.  Password has a setter, no getter.
  6. The password must be transient.

Part 2: Build a class to read/write

Write another class which will read and write user registration records to and from a binary file.  Your code must do the following:

  1. Create another Java class to test writing to a binary file.
  2. The Java class must contain a main() method.
  3. Use the console (System.in) and a Scanner to take in the target file name.
  4. Use the console and a Scanner to take in the user registration information:  remember that username, user ID and password are mandatory.  The other fields are not.
  5. Use the ObjectOutputStream class to write sample user registration records in to a file.
  6. Use the ObjectInputStream class to read sample records from the same file.
  7. Display the contents of the file, field by field, in the console output (System.out)... except the password!

Submission

  • Submit two classes
  • Submit the UserRegistration model and the reader/writer
  • Only the reader/writer should have a main() method
  • You must use object I/O to perform the reading and writing.
