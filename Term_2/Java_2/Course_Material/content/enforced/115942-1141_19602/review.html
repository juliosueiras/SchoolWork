2. A program needs to be written to play a trivia game. One of the classes being used is the TFQuestion class, which models a true-false question. This class has the following specs:

  • Attributes:
    • questionId - an integer that must be greater than 0 and represents a unique id number for each question.
    • questionText - a String that represents the question text for this question. This attribute must have a value and can't be a null-string.
    • answer - a boolean that stores the answer to the question - true or false.
  • Methods:
    • TFQuestion(int questionId, String questionText, boolean answer) - a constructor that accepts and sets each of the attributes.
    • Accessor/Mutator methods for each attribute.
    • boolean isCorrect(boolean guess) - this method accepts a boolean value representing the user's guess. If the guess is correct, the method returns true. If the guess is incorrect, the method returns false.
    • String toString() - returns a String representation of the multiple-choice question. This should show only the question I.D. and the question text formatted in the following way: 
      #XXX:  True / False :  This is the question text for this question.
  • Exceptions - mutator methods throw an IllegalArgumentException (with appropriate error message strings) if a field value is invalid.

Write a test class to test your TFQuestion class. Create a question, display the question on the screen, ask for the user's guess, and check the guess for the correct answer. Use console input for this program.
